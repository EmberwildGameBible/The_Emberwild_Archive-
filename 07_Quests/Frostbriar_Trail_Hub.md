# Frostbriar_Trail_Hub

**Origin/Provenance:** Posted by Shiverhook Outwatch at Timberline Post to keep sled routes clear.

**Cultural Tie:** Briar-bitten sledders trade favors with Saltmarrow couriers; contracts are stamped with resin thumbprints.

**Gameplay Hook:** Hub of 8 grounded quests (including 3 repeatables) guiding players through Frostbriar hazards and linking to Ashen Cry hooks.

## Quests
1. **Resin Ration Run** — Deliver briar resin to Slidecaller Ridge sledders. Repeatable. See also: [Frostbriar_Reach_World_Event](../02_Regions/Frostbriar_Reach/Frostbriar_Reach_World_Event.md), [Briar_Sinew_Binding](../08_Items/Crafting/Briar_Sinew_Binding.md), [Windbreak_Ice_Lantern](../08_Items/Trinkets/Windbreak_Ice_Lantern.md). Spawns [Resin_Ration_Tally](../09_Artifacts/Resin_Ration_Tally.md).
2. **Snow Squall Sweep** — Clear “White Howl” packs near Shatterpine Outlook. See also: [Frostbriar_Reach_Subareas_and_POIs](../02_Regions/Frostbriar_Reach/Frostbriar_Reach_Subareas_and_POIs.md), [Timberline_Sled_Rails](../08_Items/Gear/Timberline_Sled_Rails.md), [Frostbriar_Waykeeper](../03_Lore/Creatures/Frostbriar_Waykeeper.md). Spawns [Howl_Track_Report](../09_Artifacts/Howl_Track_Report.md).
3. **Briar Sap Salve** — Craft salves using frost fern resin for chill mitigation. See also: [Gnat-Smother_Salve](../08_Items/Crafting/Gnat-Smother_Salve.md), [Frostbriar_Stag_Shanks](../08_Items/Cooking/Frostbriar_Stag_Shanks.md), [Frostbite_Warden](../03_Lore/Creatures/Frostbite_Warden.md). Spawns [Salve_Order_Slip](../09_Artifacts/Salve_Order_Slip.md).
4. **Slidecaller Bells** — Trigger bells to escort a timber sled; timed repeatable. See also: [Slidecaller_Bell_Tab](../09_Artifacts/Slidecaller_Bell_Tab.md), [Frostbriar_Reach_World_Event](../02_Regions/Frostbriar_Reach/Frostbriar_Reach_World_Event.md), [Slidecaller_Key_Fragment](../08_Items/Keys/Slidecaller_Key_Fragment.md). Spawns [Bells_Run_Ticket](../09_Artifacts/Bells_Run_Ticket.md).
5. **Ash Drift Scouts** — Intercept cult envoys in Ash Drift Hollows. See also: [Ashbound_Breath_Charm](../08_Items/Trinkets/Ashbound_Breath_Charm.md), [Ashen_Cry_Grotto](../06_Systems/Dungeons/Ashen_Cry_Grotto/Dungeon_Overview.md), [Ash_Drift_Hollows](../02_Regions/Frostbriar_Reach/Frostbriar_Reach_Subareas_and_POIs.md). Spawns [Ash_Drift_Patrol_Log](../09_Artifacts/Ash_Drift_Patrol_Log.md).
6. **Timber Slide Repair** — Collect sled bolts and rails after an avalanche. Repeatable. See also: [Timberline_Sled_Rails](../08_Items/Gear/Timberline_Sled_Rails.md), [Slidebreaker_Tandem](../06_Systems/Dungeons/Ashen_Cry_Grotto/Bosses.md), [Old_Road_Circuit](Old_Road_Circuit.md). Spawns [Slide_Repair_Invoice](../09_Artifacts/Slide_Repair_Invoice.md).
7. **Elk Call Training** — Learn elk calls to soothe Frostbriar herds. See also: [Frostbriar_Stag_Shanks](../08_Items/Cooking/Frostbriar_Stag_Shanks.md), [Frostbriar_Waykeeper](../03_Lore/Creatures/Frostbriar_Waykeeper.md), [Windbreak_Bounty_Board](Windbreak_Bounty_Board.md). Spawns [Elk_Call_Sheet](../09_Artifacts/Elk_Call_Sheet.md).
8. **Slidecaller Signal Flares** — Test flare packs near Meltwater Crossings. See also: [Hookbite_Signal_Flares](../08_Items/Gear/Hookbite_Signal_Flares.md), [Meltwater_Bracer](../08_Items/Gear/Meltwater_Bracer.md), [Frostbriar_Reach](../02_Regions/Frostbriar_Reach/Zone_Overview.md). Spawns [Flare_Test_Report](../09_Artifacts/Flare_Test_Report.md).
