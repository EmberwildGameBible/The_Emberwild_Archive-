# CRAFTING GUIDE — “Sharpening Basics”

*A page from an apprentice’s workbook.*

Use slow strokes.

Angle needs to stay steady.

If it skids, reset and try again.

Check your edge by cutting paper.

If it tears, it needs more work.

If it slices clean, you’re done.

Never sharpen when tired.

— **Crafting Manual, Entry 11**
