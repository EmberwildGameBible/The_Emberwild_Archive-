# CHAPTER 15 — DUNGEONS, MINI-DUNGEONS, LAIRS & INSTANCED ENCOUNTERS

This chapter defines all structured PvE challenge content in Emberwild.

It includes:

- Lairs (open-world elite dens)
- Mini-Dungeons (2–3 player compact runs)
- Full Classic-style Dungeons (4-player)
- Open-World Boss Lairs (5–8 player zone events)
- Boss rules
- Enemy templates
- Reward structure
- Difficulty pacing
- Respawn/lockout logic
- Map placement guidelines

All systems follow Classic-era rules:

- No scaling
- No difficulty modes
- No modern systems
- Fixed enemy levels
- Handcrafted layouts
- Predictable, learnable bosses
- Real danger, real travel, real rewards

────────────────────────────────

# 15.0 — PURPOSE OF THIS CHAPTER

To define Emberwild’s group PvE ecosystem, built around:

### Difficulty Layers

- 1–2 player Lairs
- 2–3 player Mini-Dungeons
- 4-player Classic-style Dungeons
- 5–8 player Open-World Boss Lairs

### Key Outcomes

- Clear progression
- Clear challenge identity
- Classic WoW dungeon pacing
- Fixed loot tables
- Biome-themed encounters
- Realistic enemy types (creatures + humanoids)
- No supernatural mechanics

This chapter also ensures dungeons remain grounded, fair, hand-authored, and consistent with all v4.1 world rules.

────────────────────────────────

# 15.1 — CONTENT TYPE OVERVIEW

There are four major PvE structures.

────────────────────────────────

## TYPE A — LAIRS (OPEN-WORLD ELITE DENS)

Players: 1–2

Location: Open world (non-instanced)

Purpose: Short, high-risk micro-challenges

### Examples (Fully Grounded)

- Wolf dens
- Bear caves
- Boar warrens
- Serpent pits
- Tanglebrush hollows (crabs/turtles)
- Old hunter dens
- Abandoned roadside shelters
- Cliff-side cat caverns

### Features

- Small footprint
- 3–8 enemies
- 1 Rare Elite or Mini-Boss
- High material value (skins, bones, organs, claws, teeth)
- Chance for 1-in-X Ultra-Rare material caches
- No magic, no supernatural layers

### Respawn

- Normal/Elite mobs: 5–10 min
- Mini-Boss: 30–60 min

────────────────────────────────

## TYPE B — MINI-DUNGEONS (2–3 PLAYER INSTANCES)

Players: 2–3

Purpose: Compact tactical challenges

### Examples

- Abandoned watchtowers
- Flooded cellars
- Collapsed mines
- Cliffside hideouts
- Lost ranger bunkers
- Desert smugglers’ tunnels
- Forgotten burial caves (non-magical)
- Ruined frontier shelters

### Features

- 2–3 rooms
- 1 Rare Elite
- 1 Mini-Boss
- Tight corridors, short fights
- Small loot tables
- Classic WoW simplicity

### Lockout

Soft lockout: Cannot re-loot for 30 minutes after completion.

────────────────────────────────

## TYPE C — DUNGEONS (FULL 4-PLAYER CLASSIC-STYLE)

Players: 4 (Tank + Healer + 2 DPS/Utility)

Purpose: Core group PvE content

### Features

- 6–12 rooms
- 3–5 bosses (fixed order)
- Trash packs designed with pull strategy
- Environmental hazards (grounded: brush, rockfall, slippery stone, deep water, choking fog, narrow ledges)
- Profession nodes inside (ore, herbs, hides)
- Story context for each dungeon
- Classic WoW flow

### NO Scaling

- Fixed enemy levels
- Fixed boss strength
- No dynamic difficulty

### Dungeon Level Bands

- Low Dungeons: Lv 10–20
- Mid Dungeons: Lv 20–35
- Upper Mid: Lv 35–50
- Endgame: Lv 50–60

### Boss Rules (v4.1)

Bosses use:

- 2–3 mechanics max
- Frontal cones
- Heavy single-target hits
- Adds
- Environmental hazards
- Telegraphed attacks

NO:

- Enrage timers
- DPS checks
- Multi-phase cinematic fights
- MMO “dance mechanics”
- Scaling health pools

Difficulty = fair, deadly, Classic.

────────────────────────────────

## TYPE D — OPEN-WORLD BOSS LAIRS (5–8 PLAYER)

Players: 5–8

Purpose: Region-defining world events

These are not instanced — they exist in the living map.

### Examples

- Massive boar warrens
- Ancient bear dens
- Cliffside predator perches
- Highland armored beasts
- Coastal crab kings
- Large serpent hollows
- Great cat hunting grounds

### Features

- Multi-room open caverns
- Patrol groups
- Adds + elites
- A monstrous Boss (grounded, natural world boss)
- Long respawn timers
- Region-unique loot
- 1-in-X Ultra-Rare chances

### Lockouts

None — open-world Classic style.

────────────────────────────────

# 15.2 — DUNGEON STRUCTURE RULES

These rules define every dungeon and mini-dungeon in Emberwild.

### Handcrafted Only

No procedural generation.

### No Scaling

Enemies have fixed levels, fixed stats.

### Dungeon Flow (Always Classic)

1. Entrance (Low danger)
2. Early Trash (teach mechanics)
3. First Mini-Boss
4. Mid Trash (harder pulls)
5. Rare Elite Corridor
6. Second Mini-Boss
7. Final Wing (danger spike)
8. Final Boss
9. Exit path (or safe return route)

### Travel is Part of the Experience

No teleporting inside.

────────────────────────────────

# 15.3 — BOSS MECHANICS (GROUNDED v4.1)

Bosses use simple, readable, Classic-style mechanics:

### Allowed Mechanics:

- Heavy frontal swipes
- Charge → knockdown
- Patrol adds
- Summoned beasts from nearby dens
- Environmental choke points
- Fog-covered lines of sight
- Terrain hazards (rocks, water, tight corners)
- Recover phases (boss retreats briefly)

### Prohibited Mechanics:

- Magic or elemental attacks
- Phase-changing supernatural events
- DPS checks
- Timed enrages
- MMO-style AoE circles
- Teleporting bosses

Boss challenge = learning the fight, not reacting to gimmicks.

────────────────────────────────

# 15.4 — DUNGEON LOOT RULES

Dungeon loot follows Chapter 10’s rarity structure:

- Normal
- Rare
- Elite
- Rare Elite
- Mini-Boss
- Boss

### Boss Loot Contains:

- Blue-quality gear
- Profession recipes
- High-value creature materials
- Rare-crafted weapon/armor components
- Cosmetic trophies (fangs, skulls, antlers)
- Very small chance at Ultra-Rare (1-in-X)

### NO Scaling

Dungeon loot NEVER scales to player level.

### Lockout

4-player dungeons: 1-hour soft lockout

- Can re-run to help others
- Cannot re-loot until expiration

────────────────────────────────

# 15.5 — DUNGEON ENEMY AI (CLASSIC v4.1)

Enemies follow the Classic ruleset:

- Threat-based aggro (tank holds threat)
- Pack pulls (pulling one alerts neighbors)
- Leash limits (reset if kited too far)
- Patrols (roaming dangers)
- Roam clusters (family-based groups)
- Healing enemies exist but are grounded:
    - tribal shamans
    - clan medics
    - poacher field surgeons

All humanoid “casters” must be non-magical:

- smoke bombs
- fire lanterns
- snares
- traps
- morale shouts

# 15.8 — DUNGEON PROGRESSION LOOP

The intended progression:

1. Level up through homeland
2. Start with Lairs (solo/duo)
3. Mini-Dungeons (2–3 player)
4. Full Dungeons (4-player)
5. Gather gear, learn mechanics
6. Challenge Open-World Boss Lairs (5–8 player)
7. Farm Ultra-Rares, materials, profession unlocks
8. Continue cycle

This mirrors Classic WoW’s natural PvE loop, not modern MMO treadmills.

────────────────────────────────

# 15.9 — DESIGN CONSTRAINTS (v4.1 LOCK)

All dungeon content must obey:

- No difficulty scaling
- No loot scaling
- No dynamic spawns inside
- No teleportation
- No magic or supernatural mechanics
- Handcrafted layouts only
- Single static difficulty
- Classic-style enemy placement
- Fixed boss tables
- Real travel required
- Real danger (pulls matter)
- No modern system intrusions

If an idea violates ANY of these rules → it is invalid.
