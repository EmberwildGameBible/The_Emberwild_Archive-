# Hollowroot Warren — Dungeon Overview

A burrowing underforest labyrinth of root-tunnels and fungus halls.

- **Intended band:** 12–18 (normal), 18–22 (hard)
- **Key theme:** navigation pressure + resource attrition + readable lore drops
- **Entry requirements:** key fragment (optional), or faction rank (optional)

## Wings
- Wing A: Approach + mechanics tutorial
- Wing B: pressure ramp (patrols, hazards)
- Wing C: boss gauntlet + climax

## Systems hooks
- Uses Encounter Modules (ENCMOD-####).
- Uses Work Orders for repeat clears (WO-*-####).
- Drops faction tokens for vendor tiers.
