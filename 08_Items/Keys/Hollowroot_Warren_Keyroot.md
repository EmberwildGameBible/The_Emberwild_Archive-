# Hollowroot_Warren_Keyroot

**Origin/Provenance:** Living root carved into a key shape, harvested from tunnels guarded by Glass Sentinels.

**Cultural Tie:** Hollowroot guides braid family sigils into the bark before lending it to outsiders.

**Gameplay Hook:** Unlocks spore-locked side rooms and caches; using it before Vent-Siren grants bonus set pieces. Logging a use ticks [Rootbound_Path_Brief](../../09_Artifacts/Rootbound_Path_Brief.md).
