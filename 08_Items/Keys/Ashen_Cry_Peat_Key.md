# Ashen_Cry_Peat_Key

**Origin/Provenance:** Key carved from compressed peat blocks used by Shiverhook crews to seal collapsed vent doors.

**Cultural Tie:** Frostbriar locksmiths imprint Windbreak sled runes along the spine so rope teams know which bridge to reinforce.

**Gameplay Hook:** Unlocks side caches behind ashfall curtains and grants extra loot roll when used after Slidebreaker Tandem; discovery updates [Slidecaller_Field_Note](../../09_Artifacts/Slidecaller_Field_Note.md).
