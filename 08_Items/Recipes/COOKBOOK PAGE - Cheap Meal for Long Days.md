# COOKBOOK PAGE — “Cheap Meal for Long Days”

*Stained page found in a roadside inn’s kitchen.*

### **Cheap Meal for Long Days**

**Ingredients:**

- dried meat
- cabbage
- salt
- broth
- carrot (optional)

**Directions:**

Boil broth.

Add cabbage first.

Add meat when the cabbage softens.

Cook until everything looks the same color.

Cheap. Fills the stomach.

Keeps workers going without complaint.

— **Innkeeper’s Notes**
